This is the repo for the main project 


**Scope of the dissertation**

**1)** **AI-assisted Detection of OWASP API Top 10 Attacks Using Splunk SIEM**

--Research Aim : 
To investigate the effectiveness of Splunk based detection rules for selected OWASP API Top 10 attack scenarios and evaluuate whether AI-assisted alert triage can improve the effeciancy of security icident analysis.

--Scope : 
- Simulate selected OWASP API attacks(e.g., BOLA , Broken Authentication , Excessive Data Exposure)
- Collect and analyse logs using Splunk
- Develop detection rules using SPL
- Integrate an LLM-based triage component to classify alerts by sverity and recommend analysis actions
- Evaluate detection performance and alert quality. 

Why I chose this topic?
- Combines API Security,SIEM. AI and Detection Engineering.
- Strong alignment with modern SOC operations.
- High Industry relevance rather than just being a project 
- Provides measurable evaluation matricces suitable for research
